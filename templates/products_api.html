How to use
  • Use GET /api/products to fetch all products as JSON.
  • Use POST /api/add_product to add a product (admin only).
  • Use PUT /api/edit_product/<id> to edit a product (admin only).
  • Use DELETE /api/delete_product/<id> to delete a product (admin only).
Example Request (fetch all products):
curl http://localhost:5000/api/products
